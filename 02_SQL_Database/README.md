# SQL Database

Learn how to create, select, rename, and delete databases using SQL.

## Overview

A database is a structured collection of data. SQL provides commands to create and manage databases efficiently.

## Topics Covered

### 1. CREATE DATABASE
- Creates a new database
- Syntax:
```sql
CREATE DATABASE database_name;
```

Example:
```sql
CREATE DATABASE SchoolDB;
```

### 2. USE DATABASE
- Selects the database to work with
- Syntax:
```sql
USE database_name;
```

Example:
```sql
USE SchoolDB;
```

### 3. RENAME DATABASE
- Renames an existing database
- Example:
```sql
RENAME DATABASE SchoolDB TO EduDB;
```

### 4. DROP DATABASE
- Deletes a database permanently
- Syntax:
```sql
DROP DATABASE database_name;
```

Example:
```sql
DROP DATABASE EduDB;
```

## Learning Outcomes

After completing this module, you will be able to:

- Create databases
- Select databases for use
- Rename databases
- Delete databases safely
- Understand database management basics
